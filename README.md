# Ruby Pattern Matching Practice Project

This project is an exercise that was designed to help learn **Ruby pattern matching**.  
Examples:

- Array patterns  
- Hash patterns  
- Nested patterns  
- Alternatives (`|`)  
- Guards (`if`)  
- Splat matching (`*rest`)  
- As-patterns (`=> var`)  
- Pin operator (`^var`)  
- Fallback patterns
